# Rejuveta Heritage WordPress Theme

Custom WordPress theme for **Rejuveta Heritage & Conservation**.

Tagline: **Reviving Heritage. Preserving Legacy.**

## What Is Included

- Fully responsive homepage with all requested Phase 1 sections.
- Header menu with dropdowns and internal page links.
- Footer with quick links, resources, contact information, and newsletter form.
- Starter pages created automatically when the theme is activated.
- Lightweight loading animation and scroll reveal effects.
- SEO basics: title support, meta description, canonical tag, Open Graph tags, Twitter card, and NGO schema.
- Working contact form shortcode: `[rejuveta_contact_form]`.
- Newsletter form shortcode: `[rejuveta_newsletter]`.
- Backend control panel: **Rejuveta Panel** for no-code website updates.
- Editable content sections for Focus Areas, Current Initiatives, Heritage Sites, Gallery Items, and News & Events.
- Fast theme structure without page-builder dependencies.

## Installation

1. Zip the `rejuveta-heritage` folder.
2. In WordPress admin, go to **Appearance > Themes > Add New > Upload Theme**.
3. Upload `rejuveta-heritage.zip`.
4. Activate the theme.
5. The theme will create the Phase 1 pages, set the homepage, and build the main menu.

## After Activation

- Go to **Rejuveta Panel** in the WordPress admin sidebar.
- Update hero text, hero image, about text, vision, mission, phone, email, office address, social links, Google Map embed, and footer text from the panel.
- Add or edit homepage cards from **Rejuveta Panel > Focus Areas**, **Current Initiatives**, **Heritage Sites**, **Gallery Items**, and **News & Events**.
- Use each item's title, editor content, short summary, featured image or image URL, category, and custom link URL.
- Gallery categories: `sites`, `events`, `walks`, `community`.
- News categories: `news`, `events`, `workshops`, `campaigns`.
- Add your real logo in **Appearance > Customize > Site Identity**.
- Install a caching plugin such as LiteSpeed Cache, WP Rocket, or W3 Total Cache on the live site.
- Use an SMTP plugin for reliable contact form email delivery.

## Recommended WordPress Settings

- Go to **Settings > Permalinks** and choose **Post name**.
- Go to **Settings > Reading** and confirm the homepage is set to **Home**.
- Add real articles under **Posts** for News & Events and Heritage Hub content.
